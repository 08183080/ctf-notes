## fuzzing (模糊测试)

## 行业黑话
- [ ] webshell. Webshell的攻击方式通常是通过利用网站的漏洞或弱点，将Webshell文件上传到目标服务器上。一旦Webshell成功上传并执行，攻击者就可以通过它来执行各种操作，例如查看文件系统、上传和下载文件、执行命令、修改文件权限等。
